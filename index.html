Basics of the Web

The World Wide Web is a collection of billions of web pages that can be accessed through the internet. To access a web page a person will need a computer with a web browser and a connection to the internet. The computer's web browser will make a request through the internet to a server (which is just a computer that is specially configured to host web pages) by using Hypertext Transfer Protocol (HTTP). The server will send the appropriate files to the user's computer, and then the web browser will interpret and display the web page.

Web pages are created by using many different computer languages. The most important is Hyper Text Markup Language (HTML). HTML is used to define the structure and content of the web page. Cascading Style Sheets (CSS) are used to define the style and format of the web page. And JavaScript is used to define the behavior of the web page, which allows the user to interact with different components. These languages can be written in a basic text editor, although it does help to have specially designed text editor.

HTML

HTML is used to create the structure and content of a web page. An HTML document is made up of various elements, each of which serves a different purpose. For example, there is an element if you wish to include an image in the web page. There is another element if you wish to create a text header. Below is an example of the element used to create a paragraph of text content:

<p>text content for paragraph goes here</p>

Not all elements will have content, so rather than have a start and close tag they can be written with a single tag. These are called void tags.

HTML elements can also either be considered an inline element or a block element. Inline elements represent a specific section of content within an element, whereas a block element refers to an invisible box that surrounds a section of the page.

HTML elements can also have attributes, which contain additional information about the element.

DOM

The Document Object Model (DOM) refers to the 'tree-like' structure that an HTML document will undertake. This structure is formed because elements can be nested within elements. The DOM helps the web browser read the web page, and it is critical for understanding how CSS is used.

CSS

CSS is another language which has its own syntax and rules. CSS is used to apply style and formatting to the web page. CSS is particularly useful because it allows you efficiently 'cascade' a rule to many elements, thus saving you time so you are not writing redundant code. For example, you could write a single rule to turn the color of all your paragraphs red.

Final Thoughts

HTML and CSS may overwhelm you at first because there is so much that they can do - there are near endless number of elements in HTML and properities in CSS. Rather than trying to learn them all, focus on understanding how the languages work and building an awareness for what they can do.
